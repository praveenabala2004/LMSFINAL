JS Syntax

JavaScript syntax defines how programs are written. It includes rules for writing variables, functions, and control structures.

Example Program 1:


function greetUser() {
  let user = "John";
  console.log("Hello, " + user + "!");
}

greetUser();

Explanation: This program defines a function called greetUser which declares a variable user with the value "John". Inside the function, console.log() outputs a greeting message concatenating the string "Hello, " with the value of user. When greetUser() is called, it runs this code and prints "Hello, John!" to the console. This example demonstrates how to declare a function, create a variable using let, and output text to the console.

Example Program 2:


let age = 25;
if (age >= 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}

Explanation: In this example, a variable age is declared with the value 25. The if statement checks whether age is 18 or greater. If true, it prints "You are an adult." to the console; otherwise, it prints "You are a minor." This example shows how to use conditional statements and comparison operators in JavaScript syntax.

Variables

Variables store data that can be used and changed in a program.

Example Program 1:


let city = "Paris";
const country = "France";
var population = 2000000;

console.log(city + " is in " + country + " with a population of " + population);

Explanation: Here, let declares a variable city that can be reassigned. const declares a constant country that cannot be changed later. var declares a variable population with function scope (older style). The console.log prints a sentence combining these variables using string concatenation.

Example Program 2:


let temperature = 25;  // temperature in Celsius
temperature = 30;      // change temperature value

const PI = 3.14159;
// PI = 3.14;        // This will cause an error because PI is a constant

console.log("Current temperature is " + temperature + "Â°C");
console.log("Value of PI is " + PI);

Explanation: This program shows how a variable declared with let (temperature) can be reassigned later. The constant PI holds a fixed value and cannot be reassigned. Attempting to change a const variable will throw an error. The console.log statements display the current temperature and the value of PI.

Data Types

JavaScript has dynamic types. You don't need to specify the type of a variable.

Primitive data types include:

  • String
  • Number
  • Boolean
  • Null
  • Undefined
  • Symbol
  • BigInt

Example 1:


let age = 30;           // Number
let name = "Alice";     // String
let isStudent = true;   // Boolean
let unknown = null;     // Null
let x;                  // Undefined

Explanation: Variables are declared without specifying their types. JavaScript dynamically assigns the data type based on the value. age is a number, name is a string, isStudent is a boolean, unknown is null, and x is undefined because it has no assigned value.

Example 2:


// Using Symbol and BigInt
let sym = Symbol("id");
let bigNumber = 9007199254740991n;

console.log(typeof sym);       // "symbol"
console.log(typeof bigNumber); // "bigint"

Explanation: Symbol creates a unique identifier, useful for object keys. BigInt is for very large integers beyond the normal Number limits. Both have their own data types as shown by typeof.

Operators

Operators are used to perform operations on variables and values.

  • Arithmetic: +, -, *, /, %
  • Assignment: =, +=, -=
  • Comparison: ==, ===, !=, !==, >, <
  • Logical: &&, ||, !

Example 1:


let a = 10, b = 5;
console.log(a + b);    // 15 (Addition)
console.log(a % b);    // 0 (Remainder)
console.log(a > b);    // true (Comparison)

Explanation: Basic arithmetic and comparison operators used. + adds two numbers, % returns remainder, and > compares if a is greater than b.

Example 2:


let x = 5;
x += 3;  // Equivalent to x = x + 3
console.log(x); // 8

console.log(true && false);  // false (Logical AND)
console.log(!true);          // false (Logical NOT)

Explanation: Assignment operators like += simplify updating variables. Logical operators are used to combine boolean expressions: && returns true only if both are true; ! negates a boolean value.

Functions

Functions are blocks of code designed to perform a task.

They are reusable and help keep your code DRY (Don't Repeat Yourself).

Example 1:


function greet(name) {
  return "Hello " + name;
}
console.log(greet("Sam"));

Explanation: This function takes a parameter name and returns a greeting string. Calling greet("Sam") outputs "Hello Sam". Functions help organize code into reusable pieces.

Example 2 (Arrow Function):


const sum = (a, b) => a + b;
console.log(sum(5, 3));  // 8

Explanation: Arrow functions provide a shorter syntax for writing functions. Here, sum takes two parameters and returns their sum. Arrow functions are commonly used for simple functions or callbacks.

Events

JavaScript can respond to user interactions with events like clicks, mouseovers, and keystrokes.

Example 1:


document.getElementById("btn").addEventListener("click", function() {
  alert("Button clicked!");
});

Explanation: This adds a click event listener to an element with ID "btn". When clicked, an alert box appears. Event listeners allow JavaScript to react to user actions.

Example 2 (Keyboard Event):


document.addEventListener("keydown", function(event) {
  console.log("Key pressed: " + event.key);
});

Explanation: This listens for any key pressed on the keyboard and logs the key value. Keyboard events help create interactive applications that respond to user input.

DOM Manipulation

The DOM is a programming interface for HTML and XML documents.

You can access elements using getElementById, querySelector, etc.

Example 1:


let heading = document.getElementById("title");
heading.style.color = "blue";

Explanation: Access the element with ID "title" and change its text color to blue. This shows how to style elements dynamically.

Example 2 (Create and Append Element):


let newPara = document.createElement("p");
newPara.textContent = "This is a new paragraph.";
document.body.appendChild(newPara);

Explanation: This code creates a new paragraph element, sets its text content, and appends it to the end of the document body. You can dynamically add elements to the page using DOM methods.

Loops

Loops are used to execute a block of code repeatedly.

Types of loops:

  • for
  • while
  • do...while
  • forEach (arrays)

Example 1 (For loop):


for (let i = 0; i < 5; i++) {
  console.log(i);
}

Explanation: The for loop repeats the code block while the condition i < 5 is true. It logs numbers 0 to 4. Loops help automate repetitive tasks.

Example 2 (While loop):


let count = 3;
while (count > 0) {
  console.log("Countdown: " + count);
  count--;
}

Explanation: The while loop continues executing as long as count is greater than zero, decreasing count each time. It logs a countdown from 3 to 1.

Arrays

Arrays hold multiple values in a single variable.

JavaScript arrays are zero-indexed and can hold any data type.

Example 1:


let fruits = ["Apple", "Banana", "Cherry"];
fruits.push("Mango");
fruits.forEach(fruit => console.log(fruit));

Explanation: This array stores fruit names. push() adds "Mango" at the end. forEach() loops through each fruit and logs it. Arrays help organize related data.

Example 2 (Array Manipulation):


let numbers = [1, 2, 3, 4, 5];
numbers.splice(2, 1);   // Removes 1 element at index 2
console.log(numbers);   // [1, 2, 4, 5]

Explanation: splice() removes one element at index 2 (which is 3). The array updates accordingly. This method helps add or remove elements from specific positions in an array.
