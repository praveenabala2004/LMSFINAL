JavaScript Tutorial

Learn how to make your web pages interactive and dynamic!

Time remaining: 30:00

Introduction

JavaScript is the programming language of the web. Learn how it can make websites interactive.

Q1: What is JavaScript used for?

A: JavaScript is used to create interactive and dynamic web pages. It can be used for event handling, DOM manipulation, and more.

JS Syntax

Understand how to write valid JavaScript statements, blocks, and scripts.

Q1: What is the correct syntax to declare a variable in JavaScript?

A: The correct syntax is let variableName;, const variableName; or var variableName;.

Example: Using forEach() to print numbers:


const numbers = [1, 2, 3];
numbers.forEach(function(num) {
  console.log(num);
});

Variables

Learn how to declare variables using var, let, and const.

Q1: What is the difference between let and const?

A: let allows reassignment; const does not allow reassignment after declaration.

Example: Iterating over cities using let and forEach():


let cities = ['Paris', 'London', 'Tokyo'];
cities.forEach(city => {
  console.log('City:', city);
});

Data Types

JavaScript supports strings, numbers, booleans, objects, undefined, and more.

Q1: What are the primitive data types in JavaScript?

A: String, Number, Boolean, Null, Undefined, Symbol, and BigInt.

Example: Using forEach() to log data types:


let mixed = [42, 'hello', true, null];
mixed.forEach(item => {
  console.log(typeof item, item);
});

Operators

Explore arithmetic, assignment, comparison, and logical operators in JavaScript.

Q1: What is the difference between == and ===?

A: == compares values only, while === compares both value and type.

Example: Summing numbers using forEach():


let values = [10, 20, 30];
let total = 0;
values.forEach(val => total += val);
console.log('Total:', total);

Functions

Learn to create and invoke functions to structure reusable code blocks.

Q1: How do you define a function in JavaScript?

A: Use function functionName() { ... } to define a function.

Example: Using a function with forEach():


function greet(name) {
  console.log('Hello', name);
}
let names = ['Alice', 'Bob'];
names.forEach(greet);

Events

Handle user interactions using event listeners.

Q1: How do you add a click event listener?

A: Use element.addEventListener('click', function() { ... });

Example: Adding event listeners to multiple buttons:


document.querySelectorAll('.myBtn').forEach(btn => {
  btn.addEventListener('click', () => {
    alert('Button clicked!');
  });
});

DOM Manipulation

Use JavaScript to access and modify HTML elements.

Q1: How do you change the text content?

A: Use element.textContent = "New text";

Example: Changing text for all items:


document.querySelectorAll('.item').forEach(el => {
  el.textContent = 'Updated!';
});

Loops

Repeat code using for, while, or do...while loops.

Q1: How does a for loop work?

A: for (let i = 0; i < 5; i++) { ... }

Example: Using forEach() instead of a loop:


const letters = ['a', 'b', 'c'];
letters.forEach((letter, index) => {
  console.log(`Index ${index}:`, letter);
});

Arrays

Store multiple values in a single variable and work with them.

Q1: How do you declare an array?

A: let arr = [1, 2, 3];

Example: Looping through arrays with forEach():


const animals = ['Cat', 'Dog', 'Elephant'];
animals.forEach(animal => console.log(animal));
